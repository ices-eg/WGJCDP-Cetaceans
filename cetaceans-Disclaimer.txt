3. DATA SPECIFIC INFORMATION
The Vulnerable Marine EcoSystems (VME) website (https://ices.dk/data/data-portals/Pages/vulnerable-marine-ecosystems.aspx) provides the user with the ability to query the VME database and download the data records.
Download ZIP bundle includes the requested data file in the CSV format, VME Factsheet, and the data disclaimer.  
Data provided from the website are classified as either 'Public' or 'Restricted'. 
Public data are covered by the ICES Data Policy, while the Restricted data are available only on the high metadata level, with special Terms of Use defined. See the next paragraph for more information. 
Note that additional data, such as methods and quality assurance may be available. To have access to the complete data set, the user can send a request to accessions@ices.dk, specifying the download code plus the extra information required.
